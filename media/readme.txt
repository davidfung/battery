WHAT FOR
========
If you want a clutter-free way to check the battery level, this widget is for you.

Install this widget to show the battery level on your home screen.  It will refresh itself every minute of so.

EASY TO USE
===========
SHow the battery level as a percentage.  No More.  No less.  If you like minimalistic apps, you may like this widget.
